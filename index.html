PROMISE - Gestion des Congés

Application web full-stack développée dans le cadre de mon mémoire de licence pour automatiser le processus complet de gestion des demandes de congés.

Fonctionnalités Clés

  • Workflow de validation hiérarchique (Employé → Responsable → RH).
  • Calcul automatique des soldes selon le Code du travail ivoirien.
  • Interface responsive (Desktop & Mobile).
  • Notifications par email.
  • Gestion des justificatifs.

Technologies Utilisées

  • PHP
  • MySQL
  • JavaScript
  • jQuery
  • Bootstrap
